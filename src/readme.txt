This program "Timesheet Enhancer for UKG Ready" is a Chrome extension to add extra options to the Timesheet of the product UKG Ready.
UKG Ready is a product from UKG <https://www.ukg.com>. You need a licence to use it.

Manual installation for Chromium or Google Chrome
 - Unzip the package where you know you can find it again (eg, My Documents)
 - Go to chrome://extensions/
 - At the top right, turn on Developer mode
 - Click Load unpacked
 - Find and select the app or extension folder
