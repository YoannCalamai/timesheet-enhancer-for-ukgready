TimeSheet Enhancer for UKG Ready is a Chrome extension to add extra options to the Timesheet of the product UKG Ready.

UKG Ready is a Human Capital Management application from UKG (ukg.com).
You need a licence to use it. Please contact UKG for informations about licence.



This extension is NOT built by UKG. DO NOT contact UKG for support on it.



It is developed by me (Yoann Calamai) on my free time.

For bugs, feedback and feature suggestions, please contact me on the github page of the project hosted on github.com.
